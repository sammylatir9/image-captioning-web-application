1. Project title: Image Captioning Web Application 
2. Project overview: An AI-powered Image Captioning web application capable of processing images and generating captions about them using HuggingFace Large Language Model (gpt4all-falcon-Q4_0.gguf) hosted on EC2 instance on Amazon Web Services.
3. Features: - Image upload support
- Image related text generation
- HuggingFace LLM integration
- Amazon EC2 Instance hosting
4. Technologies used: - Python
- Gradio
- Transformers
- HuggingFace
- Amazon Web Services 
- GPT4all
